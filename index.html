Introduction

This class will discuss fundamental techonologies for development of real-time systems. Modern Cyber-Physical Systems are oriented from real-time systems where functional and non-functional requirements inclding timing requirements should be taken into account. Currency and parallelism are fundamental characteristics that should be considered when when rea-time systems are studied with timing requirements and constraints. In this class, we will discuss modeling, analysis and implementation of real-time systems. In addition, we will consider how to build a AI-dependent real-time systems.

Topics

  • Developmet of Cyber-Physical Systems

  • Real-Time Systems Analysis

  • Uppaal: A Model Checking Techinque for Real-Time Systems
